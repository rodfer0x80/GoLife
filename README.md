- BDD :: https://alicegg.tech/2019/03/09/gobdd.html :: https://onsi.github.io/ginkgo/

* go mod init github.com/trevalkov/gameOfLife/src
* go mod tidy
* go install github.com/trevalkov/gameOfLife/src
